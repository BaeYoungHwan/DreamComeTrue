## 목표

농장 PC(부모님 사용)에서 실제 배포를 진행하며 겪은 문제들(배치 파일 실행 불가, 단가 0원 기록)을 해결하고,
"판매는 위탁판매라 출고한다고 바로 판매되는 게 아니다"라는 핵심 피드백을 반영해 위탁 채널 판매 흐름을
출고(판매대기) → 실제 판매 수량 확정(매출 반영) 2단계로 재설계한다.

## 검증 기준

1. `python -m pytest -q` 전체 통과
2. `streamlit run app.py` 실행 후 위탁 채널 선택 시 "출고" 버튼으로 바뀌는지, 위탁 판매대기 목록에서
   수량 확정 시에만 정산 리포트에 매출이 반영되는지 브라우저로 확인
3. 직접판매/채널 미지정 판매는 기존과 동일하게 즉시 등록되는지 회귀 확인
