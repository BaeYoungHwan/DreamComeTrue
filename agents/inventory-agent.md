## 역할

재고·판매(정산) 로직 전담 — 품목, 입출고, 매출 계산의 일관성을 담당

## 담당 영역

- 재고는 음수가 될 수 없다 (출하량이 재고량을 초과할 수 없음)
- 모든 입출고는 품목+규격 단위로만 기록한다
- 매출은 출하처·단가·수량이 모두 있어야 확정된다

## 담당 파일

- src/inventory/
- src/sales/

## 작업 범위

- 담당 도메인 파일만 수정
- 도메인 외 파일은 읽기 전용
- 다른 도메인 에이전트와 직접 통신하지 않음 — Coordination Lane 경유

- 변수명·함수명: 영어
- 주석·커밋 메시지: 한국어
- type hint 필수 (Python), JSDoc 권장 (JS/TS)
